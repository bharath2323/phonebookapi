PHONEBOOK
A phonebook API developed using the HTML CSS in front end and by NODEjs
at backend. Basically one can add a contact to the phonebook database
which takes the attributes : name,gender,mobile,email

the zip file contains the html and css files. The main nodejs file named render.js is the root of this API
the database is created using XAMPP control protocol. 
One must install package.json,mysql in the nodejs modules
